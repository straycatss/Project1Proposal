Q-ARCHIVE website early visualisation

The website will have a modern, uncluttered layout.

The main focus of the page is the map, where the users can navigate around Wellington city and find where QR codes have been documented.

These locations are marked with a location symbol with the timming pattern Q symbol from the logo. Users can click on these to bring up more information about the specific code including photos and a hyperlink to the URL.
